# Logo assets

- `logo.png`: original 1536x1024 source artwork.
- `logo.svg`: scalable wrapper using the original artwork.
- `logo-mark.png`: cropped 768x512 UI mark.
- `logo-mark-{32,64,128,256,512}.png`: responsive UI sizes.
- `logo-mark.webp`: optimized UI image.
- `logo-icon-{32,64,180,192,512}.png`: transparent square app and favicon sizes.
- `logo-full-768.png`: reduced full-frame artwork.
- `logo-social-1200x630.{png,webp}`: Open Graph and social sharing images.
